This folder contains some tools required to build, test, and other small utilities used by this project. Those tools usually require some installation.


================================================================================
                          REQUIRED TO GENERATE RELEASE
================================================================================

[ Visual Studio ]
USE: Compile project files into executable.
URL: http://msdn2.microsoft.com/fr-fr/vstudio/default.aspx
VERSION USED: Visual Studio 2005
INSTALLATION: Use setup.

[ NSIS ]
USE: Generate the insallation program.
URL: nsis.sourceforge.net/
VERSION USED: 2.3
INSTALLATION: Use setup.

================================================================================
                           REQUIRED FOR DEVELOPMENT
================================================================================

[ Python (or Perl) ]
USE: Generate the unit tests runners.
URL: www.python.org/download/
VERSION USED: 2.5
INSTALLATION: Put "python" in the PATH.


================================================================================
                                   HELPERS
================================================================================

[ LineCounter++ ]
USE: Count number lines in the source code.
URL: N/A (custom modified version of LineCounter)
INSTALLATION: None.

[ Beyond Compare ]
USE: Compare differencies between files/folders.
URL: http://www.scootersoftware.com/
INSTALLATION: Use setup.

[ Visual Assist X ]
USE: Code helpers for Visual Studio.
URL: http://www.wholetomato.com/downloads/default.asp
INSTALLATION: Use setup after install VS2005.
