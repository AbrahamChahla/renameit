Most of the tasks can be done by using the wizards. Sometimes you have to use two wizards (or the same twice) to achieve what you want. All the wizards help you to create a Renamer filter for a specific task.

To use a wizard, click on the arrow right of the the "Renamer" button, or left click on that button. Most wizards are self explanatory.

Even if they help you to create the filter, they don't replace it: Once it is created, if you want to modify the filter you'll only be able to modify the more complex generated Renamer filter.

After some practice you should be able to understand the full Renamer filter and use it to create more advanced renaming rules.
