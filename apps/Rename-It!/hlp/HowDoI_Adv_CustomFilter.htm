When the Renamer filter is not enought to fulfil your needs, you can create a console application that will output (in plain text in the console) the info you need. Your application can take as argument the file/folder name, the filtered part and so on, as you like.

A full example is provided with Rename-It! in the "External processor sample" folder in the folder you've installed Rename-It!. The C source code of the custom application is provided also. To see the example, add some images to the rename list and load the preset filters of the sample folder.
