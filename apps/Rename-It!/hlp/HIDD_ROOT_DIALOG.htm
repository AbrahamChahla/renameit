Rename-It! can rename many files, folders or even a full path a file using renaming filters. It may sound complicated at first, but after a few times you'll be able to rename files in seconds and in only a few clicks.

To start renaming some files or folders, open the Windows Explorer, select a bunch of files or folders, right click on them to open the classical context menu, and choose Rename-It!. All files, folders, and the files in those folders will be added to the files list of Rename-It!. This list, visible at the bottom of the main window, contains the files and folders to be renamed. Alternatively you can drag&drop files/folders to the Rename-It! window.

Once you have a set of files/folders to be renamed you have to define what part you would like to rename. Select what part to rename by highlighting it in the "Rename what" text box. The filters will only affect that part and warn you if you attempt anything that would compromize this.

To rename files' names only, start by selecting "Filename" in the "Rename what" box. Then add renaming filters. Each filter will be applyed to each file name (and only to the files' names) sequentially. The top filter is the first executed (see flowchart). The filtered part name will then be passed to the other filters, if any.

To rename folders only it's the same procedure than renaming files but you select "\..\Folders\" in the "Rename what" box. Note that the files in the folders disappear from the preview to only display unique folder names.

Each time you add/modify/delete a filter the preview list will show a before and after renaming preview.

Once you are ready, just press the "Rename" button. If problem occur a window will appear for you to correct them before the renaming start. After that, just close Rename-It!.
