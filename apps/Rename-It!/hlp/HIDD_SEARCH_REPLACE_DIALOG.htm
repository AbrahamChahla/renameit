The Renamer filter is the root of Rename-It!. It's an advanced search and replace function with advanced tools like regexp, enumerations, ID3 Tags, case changer.

The basic function is to enter a search string in the search box, and a replacement string in the replacement box. Each time the string is found in the part behind renamed, it's replaced. You may check different search or replacement options and use more advanced search & replace with wildcards or regexp.

For wildcards, put a ? to match any single character or * to match any character(s) (0 or more) in the search box. Example: Searching "my_*_name" would match a file named "my_file_name". In the replacement box you can also use *; it'll be replaced by the matched string. Example: Replacing by "new_*_name" in our previous example would finally give "new_file_name".
