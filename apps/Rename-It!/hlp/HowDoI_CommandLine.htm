You can use Rename-It! in a batch to provide an automated advanced renaming.

Define some filters and save them. You should add some comments to help you remember what the filters do. To rename some files/folders using those preset filters, execute:

RENAMEIT /X filter_file path [/R] [/A] [path [/R] [/A] [...]] [/LOG log_file [/V]]

/X filter_file Path to your saved filter file (*.rit).
path Path to a file or a folder to be renamed (can contain wildcards characters ? or *). The /R and /A are options that can be put after the path.
/R Include the folders and sub-folders of the previous path.
/A Include all files, even the system files of the previous path.
/LOG log_file Don't stop on warnings or errors but log them in a file named log_file instead.
/V Also include files correctly renamed in the log file.

Sample 1: Rename-It!.exe /x my_filter.rit image*.jpg

Sample 2: Rename-It!.exe /x ".\my_id3_filter.rit" "C:\My music folder\" "..\a_file.mp3" "D:\Zic\"
